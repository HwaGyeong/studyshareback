### 📕 Main Function

- **메인 화면**
	- 로그인 및 회원 가입
    - 피드 조회
	
- **Todo List**
	- 개인별 투두 리스트 작성 
    - 날짜별로 등록, 매일 등록

- **피드 조회**
	- 글 작성, 삭제, 수정, 조회, 좋아요
    - 사진업로드, 멘트 업로드, 댓글

<br>
